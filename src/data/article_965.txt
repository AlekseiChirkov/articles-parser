Автор: Mojsha

Теги: Блог компании SkillFactory, Python *, Программирование *, API *,
Лайфхаки для гиков

Контент: Написанная с помощью PyFilesystem функция поиска дубликатов файлов
будет работать без изменений с жёстким диском, zip-файом, FTP-
сервером, Amazon S3 и т. д., этот API абстрагирует от физического
расположения файла. В нём меньше способов выстрелить себе в ногу, чем
у модулей os и io. Руководством из документации делимся к старту курса
по Fullstack-разработке на Python.
