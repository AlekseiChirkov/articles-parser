Контент: Согласно web.dev, улучшение качества взаимодействия с пользователем —
ключ к долгосрочному успеху любого сайта. Понимая, насколько важен для
развития интернета качественный пользовательский опыт, компания Google
в 2020 году представила инициативу Web Vitals. Это единое руководство
по метрикам качества, которые необходимы для обеспечения удобного
взаимодействия с пользователем. Хотя за прошедшие годы Google создала
ряд инструментов для измерения эффективности взаимодействия и
составления отчетов (Lighthouse, PageSpeed Insights и Measure), Web
Vitals должна упростить измерение производительности и помогает
разработчикам сайтов сосредоточиться на самых важных показателях —
Core Web Vitals.
