Название: Java 17: Pattern Matching for switch

Автор: dbudim

Дата: 2021-09-21, 15:52

Теги: Программирование *, Java *, API *

Контент: 14-го сентября состоялась презентация Apple, в этот же день произошло
не менее важное событие - релиз Java 17. Среди новых фич подъехал
паттерн матчинг для switch в preview моде.История началась с того, что
в jdk 16 расширили instanceof оператор, который теперь может принимать
type pattern и выполнять матчинг по паттерну. Это маленькое изменение
позволило упростить типичную конструкцию с проверкой на тип и
последующее приведение.
